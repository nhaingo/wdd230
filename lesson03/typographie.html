Typographie

Heading Font: Otomanopee One, Times New Roman, serif.

I choose Otomanopee One because it has a modern look and has a warm feel. It is also easy to read. As a heading font, it has an attractive look and catch promptly the reader attention, so he/she can notice where to start. Otomanopee One is supoorted by all browsers and is a web safe font.

Paragraph Font: Montserrat, serif

Montserrat is a versatile font and can be used in multiple domains, one of the reason I chose to use it for my paragraph. It gives a friendly look to my website. I like it also because it is easy to read. All the characters have the same width. This font suits more my website main purpose and is supported by many browsers..
